## üì• **Installation & Setup**

### **Step 1: Install the Theme**

1. Go to **WordPress Admin** ‚Üí **Appearance** ‚Üí **Themes**
2. Click **"Add New"**
3. Click **"Upload Theme"**
4. Upload the `code95.zip` file
5. Click **"Install Now"**
6. Click **"Activate"**

### **Step 2: Import Demo Content**

#### **Method 1: WordPress Import (Recommended)**

1. Go to **WordPress Admin** ‚Üí **Tools** ‚Üí **Import**
2. Click **"WordPress"** (install if not available)
3. Click **"Run Importer"**
4. Upload the `demo-content/code95.WordPress.2025-08-02.xml` file
5. Click **"Upload file and import"**
6. Map authors to existing users or create new ones
7. Click **"Submit"**

#### **Method 2: All-in-One WP Migration Plugin**

1. **Install Plugin:**
   - Go to **WordPress Admin** ‚Üí **Plugins** ‚Üí **Add New**
   - Search for **"All-in-One WP Migration"**
   - Click **"Install Now"**
   - Click **"Activate"**

2. **Import Site:**
   - Go to **WordPress Admin** ‚Üí **All-in-One WP Migration** ‚Üí **Import**
   - Click **"Import from"** ‚Üí **"File"**
   - Upload the `.wpress` file (if available)
   - Click **"Import"**
   - Wait for the import to complete

- **Username**: `admin`
- **Password**: `admin`
